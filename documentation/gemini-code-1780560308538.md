 Modul Pembayaran (Verifikasi Manual): Menampilkan daftar seluruh konfirmasi 
pembayaran serta dokumen/foto bukti transfer yang diunggah oleh siswa. Admin memegang 
kendali penuh untuk menekan tombol aksi Approve (Setujui) atau Reject (Tolak). Jika 
disetujui, sistem otomatis menambah kuota paket atau sesi belajar pada akun siswa terkait. 
• Modul Penjadwalan Terpusat (Admin-Controlled): Pengelolaan data fundamental kelas 
seperti paket belajar, nama kelas, mata pelajaran, dan tarif penentuan sesi. Menyediakan 
visualisasi Kalender Jadwal interaktif untuk memetakan kombinasi Siswa + Tutor + 
Ruang/Media + Jam secara real-time guna mengeliminasi risiko jadwal bentrok. 
• Modul Evaluasi & Laporan: Berfungsi memvalidasi absensi harian yang dimasukkan oleh 
pihak Tutor. Admin memiliki hak publikasi (Publish) untuk merilis laporan nilai kuis, tugas, 
serta catatan performa belajar siswa sehingga dapat dilihat langsung di halaman orang tua. 
• Manajemen Biodata Tutor: Halaman mandiri bagi pengajar untuk melengkapi info profil, 
keahlian mata pelajaran yang diampu, serta memperbarui nomor kontak aktif. 
• Kalender Jadwal Mengajar: Fitur monitoring jadwal dalam format kalender berkala 
(mingguan/bulanan) yang bersifat read-only, menampilkan sesi mengajar yang telah 
ditetapkan oleh tim Admin. 
• Form Evaluasi Kelas: Fasilitas input pasca-kelas untuk mengisi daftar kehadiran (absensi) 
siswa, memasukkan skor nilai tugas/kuis harian, serta ruang narasi teks untuk catatan 
perkembangan akademis maupun perilaku anak.
